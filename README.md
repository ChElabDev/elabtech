# E-lab Tech Inc. Portfolio

Hello, this is our portfolio project. This is a simple portfolio project that we have created for our inc.
In this portfolio, you will find a list of our projects, testimonial area from our clients, and our contact form. Fell free to contact us if you have any questions or want to work with us.

## Technologies used

- [React.js](https://pt-br.reactjs.org);
- [Styled-components](https://styled-components.com/);
- [Typescript](https://www.typescriptlang.org/);
- [Material UI](https://mui.com/);
- [Material Icons](https://mui.com/components/material-icons/);


## Getting Started

First, run the development server:

```bash
npm run dev
# or
yarn dev
```
